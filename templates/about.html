About Green Marketplace

The Green Marketplace is an online platform dedicated to promoting eco-friendly shopping. This innovative marketplace connects buyers and sellers of environmentally sustainable products, fostering a community focused on green living. The platform allows users to buy, sell, and trade a wide range of eco-friendly items such as reusable goods, sustainable fashion, organic products, and more.

Built using Django, Python, and PyCharm, the Green Marketplace offers robust user authentication features, including login, logout, and password recovery. Registered users can list items for sale, upload product images, and provide detailed descriptions, while unregistered users can browse the listings with limited access. An intuitive search bar enables users to find products by category, price range, and keywords, enhancing the shopping experience.

To ensure a seamless user experience, the Green Marketplace employs a visually appealing design with responsive layouts using Bootstrap. The database is managed in JSON format, with initial data loaded through fixtures. Class-based views streamline the code structure, and regular progress is tracked and managed via GitHub. Additional features include user ratings, reviews, and wish list.

The Green Marketplace not only facilitates sustainable shopping but also educates and empowers users to make environmentally conscious choices, contributing to a greener future. This project demonstrates a commitment to environmental sustainability and leverages modern web development tools and practices.
